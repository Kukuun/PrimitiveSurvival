# PrimitiveSurvival

Made with: VS console application and C++.

A little, primitive survival game I made some time ago.
  
Instructions:
  - W, A, S, D for movement.
  - Avoid hostile boxes.
  
Dependencies:
  - OpenGL.
  - (Free)glut.
  - IrrKlang.
  - SOIL.

The amount of time you survived is stored in the _RoundTime.txt file.
